Adds a new window/tab to manage custom CSS, class and id fields.

== Description ==

Allows you manage and change all custom CSS , class and id fields from the current elementor editor page on a seperate window/tab.


You can support development under
https://github.com/RTO-Websites/elementor-custom-css-quick-access

== Installation ==

Just download, install and activate

== Documentation ==

In the bottom navbar of the Elementor editorpanel a new button will appear to open and close Elementor Custom Css Quick Access
